> ES6 允许按照一定模式，从数组和对象中提取值，对变量进行赋值，这被称为解构（Destructuring）
```
let [a, b, c] = [10, 20, 30];
    
```
1. 数组解构
2. 默认值
```
let [a = 10, b] = [1, 20];
    console.log(a, b);
```
> 可以使用reset操作符... 但是只能当作做最后变量使用
3. 对象解构
```
{
    key:变量名
} = {
    key:变量值
}

```
4. 字符串的解构赋值
5. 函数参数的解构赋值
用途：
6. 交换变量的值
7. 从函数返回多个值
8. 函数参数的定义
9. 提取 JSON 数据
10. 函数参数的默认值
